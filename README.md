## 1.使用说明

    go get -u github.com/feelingsray/Ray-Utils-Go

## 2.主要功能

+ 协程管理器
+ 多种编解码
+ http中间件
+ 客户端连接
+ 通用工具包
+ 日志管理器

## 3.目录说明

    Ray-Utils-Go
    ├── appManager          协程管理
    ├── encode              编解码器
    ├── logger              日志管理
    ├── middleware          http中间件
    ├── minio               对象存储
    ├── mongo               Mongo连接
    ├── netHepler           IP帮助
    ├── newMongo            Mongo连接
    ├── secret              加密工具
    ├── tools               工具软件

## 4.go版本 >=1.18

    > go version
